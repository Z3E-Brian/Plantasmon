    **IMPORTANT:** All mock variables (`mockGetCurrentUserId`, `mockGetUserActivities`, `mockToActivityData`) MUST be declared outside and above the jest.mock calls. Since jest.mock is hoisted, use this pattern:
    
    ```typescript

    const mockGetCurrentUserId = jest.fn();
    ```

    ```typescript
    // Define mocks before any jest.mock calls (they will be hoisted)
    const mockGetUserActivities = jest.fn();
    const mockToActivityData = jest.fn();
    ```
