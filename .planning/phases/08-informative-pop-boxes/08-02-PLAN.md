  <action>
    Create `src/components/ui/InfoBottomSheet.tsx` with the following specification.

    **Prop interface:**
    ```tsx
    interface InfoBottomSheetProps {
      visible: boolean;
      title: string;
      message: string;
      icon?: string;         // emoji string like "🌱", "📸", "🔍"
      showDontShowAgain?: boolean;  // shows "No mostrar de nuevo" checkbox (default false)
      onDismiss: () => void;
      onDontShowAgain?: () => void; // called when user checks + dismisses with permanent flag
    }
    ```

    **Implementation:**
    - Use `Modal` from `react-native` with `transparent={true}` and `animationType="none"` (we control animation manually)
    - `visible` prop controls Modal visibility
    - **Dark overlay:** `View` with `position: 'absolute', top: 0, left: 0, right: 0, bottom: 0, backgroundColor: 'rgba(0,0,0,0.5)'`. Per D-07.
    - **No tap-outside dismiss:** The overlay `View` does NOT have an `onPress` handler. Per D-09.
    - **Bottom sheet container:** `View` positioned at bottom with `borderTopLeftRadius: 24`, `borderTopRightRadius: 24`, `backgroundColor: COLORS.card` (#163d28), `padding: 24`, `paddingBottom: 36` (extra for safe area)
    - **Slide-up animation:** Use `useRef(new Animated.Value(0)).current` as `slideAnim`. When `visible` becomes true, run `Animated.spring(slideAnim, { toValue: 1, useNativeDriver: true, damping: 20 })`. Map to translateY: `slideAnim.interpolate({ inputRange: [0,1], outputRange: [400, 0] })`. Per D-06.
    - **When visible becomes false:** Reset slideAnim to 0 immediately (no animation needed, Modal hides instantly)
    - **Swipe-down-to-dismiss:** Use `PanResponder` from `react-native` on the sheet container:
      - `onMoveShouldSetPanResponder: (_, gs) => Math.abs(gs.dy) > 10 && gs.dy > 0` (only vertical downward)
      - `onPanResponderMove: (_, gs) => { if (gs.dy > 0) slideAnim.setValue(1 - gs.dy / 400) }` (clamp to positive dy)
      - `onPanResponderRelease: (_, gs) => { if (gs.dy > 120 || gs.vy > 0.5) { onDismiss(); } else { Animated.spring(slideAnim, { toValue: 1, useNativeDriver: true }).start() } }`
      - If swiped far enough (>120px or fast velocity), dismiss. Otherwise spring back. Per D-08.
    - **Layout:**
      - Header row: `icon` emoji as large Text (fontSize: 40) if provided, then `title` as Text with `fontSize: 20, fontWeight: '700', color: COLORS.foreground`
      - Body: `message` as Text with `fontSize: 15, color: COLORS.cardForeground, lineHeight: 22, marginTop: 12`
      - If `showDontShowAgain` is true: a row with a TouchableOpacity that toggles a local `[dontShowAgain, setDontShowAgain]` state. Show a checkbox-style View (border 2px, w/h 22, borderRadius 4, filled with COLORS.primary if checked). Label: "No mostrar de nuevo" in `color: COLORS.cardForeground, fontSize: 14`.
      - Bottom button: TouchableOpacity with `backgroundColor: COLORS.primary, borderRadius: 12, paddingVertical: 14, alignItems: 'center', marginTop: 20`. Text: "Entendido" with `color: COLORS.primaryForeground, fontSize: 16, fontWeight: '600'`.
      - Button onPress: if `dontShowAgain && onDontShowAgain`, call `onDontShowAgain()` first. Then call `onDismiss()`.

    **StyleSheet** at bottom of file for all styles. Do NOT use inline styles.

    **Default export:** no. Named export: `InfoBottomSheet`.

    **Key behaviors:**
    - Swipe down → dismiss (simulates the drag handle on bottom sheets)
    - No tap-outside → does NOT dismiss (per D-09)
    - "Entendido" → calls onDismiss (which screen uses to set per-session dismissal per D-04)
    - "No mostrar de nuevo" checked + "Entendido" → calls onDontShowAgain (which screen uses to set permanent dismissal per D-05)

    **IMPORTANT:** Do NOT use Radix primitives or any web-only libraries. This is a pure React Native component using Modal, Animated, PanResponder, View, Text, TouchableOpacity, StyleSheet. No import from `@/src/lib/utils` or Radix.

      ```
      Animated.parallel([
        Animated.spring(slideAnim, { toValue: 1, useNativeDriver: true, damping: 20 }),
        Animated.spring(scaleAnim, { toValue: 1, useNativeDriver: true, damping: 12, stiffness: 200 }),
      ]).start()
      ```
    - The icon gets a scale transform: `transform: [{ scale: scaleAnim }, { translateY: slideAnim.interpolate(...) }]`
    - Same swipe-down-to-dismiss as InfoBottomSheet
    
    **Layout:**

    - No "No mostrar de nuevo" checkbox (celebrations are one-time events per unlock)
